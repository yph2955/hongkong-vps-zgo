**第一，线路。** 这是香港VPS的灵魂。香港到内地的物理距离确实近，但"近"不等于"快"。如果走的是普通国际线路，数据包可能先绕到新加坡、再绕到日本，最后才进国内——物理距离的优势全被绕没了。能称得上"稳定"的香港VPS，至少得具备三网直连或BGP优化线路，电信走CN2、联通走9929、移动走CMI/CMIN2，这才是正经的回国高速公路。
